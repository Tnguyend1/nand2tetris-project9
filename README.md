# Project 9: Catch The Dot

This is a custom Jack game for nand2tetris Project 9.

## Idea
- You control a square player.
- A target dot keeps moving.
- Touch the target to score points.
- Press `Q` to quit.

## Controls
- Up arrow: move up
- Down arrow: move down
- Left arrow: move left
- Right arrow: move right
- `Q`: quit game

## Files
- `Main.jack`: program entry point
- `Game.jack`: game loop, input handling, scoring
- `Dot.jack`: drawable/movable square object

## How to Run
1. Open the nand2tetris VM Emulator.
2. Load all `.vm` files generated from these `.jack` files (compile first with your Jack compiler).
3. Run the program.
